# RentSplit-Privacy-file
A clean, privacy-focused iOS app built with SwiftUI to help roommates split rent and shared household expenses using high-precision math
 Privacy Policy for RentSplit

**Effective Date: April 29, 2026**

RentSplit was built as a local-only utility tool. This app does **NOT** collect, store, or share any personal user data.

* **Data Collection:** All household data and expense records are stored locally on your device using persistent storage. This data stays on your device and is not accessible by the developer or any third party.
* **Data Storage:** All household data and expense records are stored locally on your device.
* **Third Parties:** No data is shared with third-party services or trackers.
